# cph_0526

一個存放庫，有多個資料夾以及專案

## 專案結構

```
cph_0526/
├── api/              # API 資料夾（包含多個版本）
│   ├── v1/          # API 版本 1
│   └── v2/          # API 版本 2
├── docs/            # 文件資料夾
├── projects/        # 專案資料夾
│   ├── project1/    # 專案 1
│   └── project2/    # 專案 2
└── shared/          # 共享資源
    ├── utils/       # 共用工具
    └── config/      # 共用設定
```

## 資料夾說明

- **api/** - 包含不同版本的 API 實作
- **docs/** - 存放專案相關文件
- **projects/** - 包含多個獨立專案
- **shared/** - 存放所有專案共用的程式碼和設定

## 開始使用

請參考各個資料夾內的 README.md 了解更多詳細資訊。
